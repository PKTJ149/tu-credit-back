# Finance and Payment UX Copy Draft v1

## Purpose

This document provides first-pass UX copy for the finance/payment redesign.

It is intended as working draft content for:

- wireframes
- high-fidelity design
- prototype reviews

## Voice

- Clear
- Reassuring
- Official
- Transaction-focused
- Direct

Avoid:

- vague status language
- bureaucratic jargon
- emotionally cold rejection copy

## Registration Confirmation

### Title

- `Registration complete`

### Body

- `Your registration has been recorded successfully. Review the next step below to continue.`

### Payment Required Variant

- `Payment is required before this registration can move forward.`

### Primary CTA

- `Go to finance`
