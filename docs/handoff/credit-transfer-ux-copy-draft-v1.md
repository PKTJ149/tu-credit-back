# Credit Transfer UX Copy Draft v1

## Purpose

This document provides first-pass UX copy for the credit transfer redesign.

It is intended as working draft content for:

- wireframes
- high-fidelity design
- prototype reviews

## Voice

- Clear
- Reassuring
- Academic
- Supportive
- Structured

Avoid:

- vague bureaucratic wording
- unexplained academic jargon
- internal staff-centric language as primary UX copy
